Installing R/RStudio

To use R and RStudio, you need 2 things:

  1. R
  2. RStudio

If for any reason you can't install R/RStudio right now or you would like to use a temporary version to experiment with, you may use this R Sandbox. Please note that the sandbox is intended as a temporary solution and may take a while to load, may not always be available, and will not save your work.

Install R

  1. Go to this website to download R.

    Note: R is downloaded from websites called "CRAN mirrors." These "mirror" websites are copies of a master website that contains the latest version of R. CRAN stands for "Comprehensive R Archive Network" and refers to the entire group of mirror sites with copies of R. Dividing the distribution of R between these mirror sites helps reduce traffic, improve access speed, and ensure ____. If the CRAN mirror linked above does not work, please choose a new CRAN mirror here.

  2. Click on the download link for your operating system.
  3. Next step.

Install RStudio

  1. Go to this website to download RStudio.
  2. Click on the download link for your operating system.
  3. Run the downloaded installer and follow the instructions it provides.

There are 4 types of functionality:

  1. Basic calculations
  2. Built-in functions
  3. Custom functions
  4. Someone Else's Custom functions (Libraries)

1. Basic Calculations

The simplest functionality that R has is the ability to do basic calculator math using operators such as +, -, *, /, and ^.

You can try this for yourself by typing a basic calculation such as 1 + 1 in the console and pressing enter to run it. You should see an output like so:

> 1 + 1
[1] 2

In R, spacing doesn't matter so you will get the same answer no matter where you put the spaces.

The next level of functionality are built-in functions such as:
